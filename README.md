Adtrafik SEO QA Tool
======================

Greasemonkey/Userscripts extension for QA testing:
* Canonical Tag
* META Tags (Check & Length)
* Pressence of Analytics and other Tracking Scripts
* Header Info

Only Works on the Following Domains:
* PokerStars (Live/QA)
* Fulltilt (Live/QA)
